Contact Form Project

A responsive, full-stack contact form for collecting and storing user messages securely.

This project allows users to submit their name, email, phone number, and message through an intuitive contact form. The form validates inputs on both the client and server sides. Submissions are stored in a MongoDB Atlas cloud database, and the backend is built with Express.js and Node.js. It’s designed to be integrated into a larger portfolio or business site, and the user interface is clean and professional.

Technologies Used: HTML, CSS, JavaScript, Node.js & Express.js, MongoDB Atlas, Render (for backend deployment)

CNN Model for Waste Classification

A machine learning model that classifies plastic waste images into various categories using CNN.

This project involves building a Convolutional Neural Network (CNN) to identify and classify different types of plastic waste, which is crucial for waste management and recycling automation. The model is trained on a dataset of plastic waste images and deployed using Streamlit for easy demo access. It demonstrates my understanding of deep learning, image processing, and real-world sustainability solutions.

Technologies Used: Python, TensorFlow / Keras, NumPy, Matplotlib, Streamlit (for deployment)
